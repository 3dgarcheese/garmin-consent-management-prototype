# Garmin Consent Management Prototype

A full-stack web application developed as part of a Software Engineering course project that demonstrates secure management of user consent for wearable health-data sharing.

## Overview

This prototype allows users to create, update, and revoke consent preferences for sharing health information collected by wearable devices. The project demonstrates full-stack web development principles using JavaScript, Node.js, Express, HTML, and CSS.

## Features

- User consent creation
- Consent modification
- Consent revocation
- Interactive web interface
- Backend request handling
- Client-server communication
- Responsive user interface

## Technologies

- JavaScript
- HTML5
- CSS3
- Node.js
- Express.js

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/garmin-consent-management-prototype.git

cd garmin-consent-management-prototype/backend

npm install

npm start
```

Open the frontend in your browser.
